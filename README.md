# ⚡ Zyan Scraper

> Zyan Scraper adalah REST API berbasis Node.js yang digunakan untuk mengambil (scraping) berbagai data dari internet seperti anime, jadwal sholat, downloader, dan tools lainnya secara cepat, ringan, dan mudah digunakan.

---

## 🚀 Fitur Utama

- 🎬 Pencarian Anime  
- ☪️ Jadwal Sholat (API Islami)  
- 📥 Downloader Tools (akan dikembangkan)  
- ⚡ Ringan dan cepat  
- 🧩 Struktur modular (mudah dikembangkan) 
- 📂 Dan masih banyak lagi 
- 🤖 Cocok untuk bot WhatsApp, Telegram, dan website
